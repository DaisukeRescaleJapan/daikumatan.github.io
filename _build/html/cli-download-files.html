3. 計算結果のダウンロード (ファイル名を指定する方法)¶

3.1. 前提¶

下記を実施済みであること

  • RescaleCLIのセットアップ方法

注釈

rescale.jar の保存先が /usr/local/bin/rescale.jar であることに注意してくださ

3.2. スクリプトのダウンロード¶

以下より rescalecli-download をダウンロードしてください。

src/rescalecli-download

ダウンロードしたスクリプトに実行権限を付与します。

chmod +x rescalecli-download

3.3. スクリプトの実行方法¶

対象とするジョブIDとダウロンドーロするファイル名を指定します。複数ファイルも指定することができます。ジョブIDが分からない方は “Rescale の ジョブID とは” をご参照ください

JOB_IDの指定¶
JOB_ID='XXXXXX'
ダウンロードしたいファイルネームの指定¶
FILE_NAME='myFile1 myFile2 *.log'
スクリプトの実行¶
./rescalecli-download ${JOB_ID} ${FILE_NAME}

#!/bin/sh
JOB_ID=$1
FILE_NAME=$2

JP="https://platform.rescale.jp/"
OPTION='sync'
java -jar /usr/local/bin/rescale.jar \
-X ${JP} ${OPTION} \
-p ${RESCALE_API_TOKEN} \
-j ${JOB_ID} \
-f ${FILE_NAME}
